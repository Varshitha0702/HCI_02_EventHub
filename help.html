Event Hub Help

Welcome to the Event Hub Help Page

At Event Hub, we provide an easy-to-use platform for managing and viewing your events. Whether you're a first-time user or a regular, this page will guide you through how to use the Event Hub, and help you troubleshoot any questions you may have.

Key Features & Functionality

Here are some of the key features that make Event Hub easy to use and efficient for managing your events:

  • Interactive Event Calendar: View your events on a user-friendly calendar layout. Easily switch between months and years with the calendar navigation.
  • Search Events: Quickly find events using the Search Bar that allows you to search by event name, date, or category.
  • Add & Manage Events: Add new events directly from the Event Calendar interface. You can set the date, time, location, and description of the event.
  • Event Reminders: Receive notifications and reminders for upcoming events directly through the app.
  • Yearly View: Switch to the yearly view to plan events throughout the year at a glance.
  • Event Categories: Organize your events by categories like 'Workshops', 'Meetups', 'Conferences', etc.
  • Event Notifications: Get email or push notifications for events you're subscribed to.

How to Use Event Hub

Follow these steps to get the most out of Event Hub:

  1. Navigate to the Event Calendar page to view upcoming events.
  2. Use the Year Selector to browse events for any year.
  3. Click on a specific date to view detailed events for that day.
  4. Search for events using the Search Bar at the top of the page.
  5. To add a new event, click the Add Event button located in the event management section.
  6. For recurring events, select the Recurring Event option when adding your event.

Frequently Asked Questions (FAQ)

How do I add a new event?

To add a new event, click the Add Event button on the Event Calendar page. You will need to enter the event name, date, time, and location. Once submitted, the event will appear on the calendar.

Can I view events for any year?

Yes, you can select a year from the dropdown menu located at the top of the calendar to view events from that year. The calendar will automatically adjust to show the events of the selected year.

How can I search for specific events?

You can use the Search Bar located in the header of the page to search for events by name, date, or category. Just type in the event keyword, and relevant results will appear.

How do I contact support?

If you need assistance, you can reach out to our support team via the Contact Us section, or email us at support@eventhub.com.
